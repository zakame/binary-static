登录故障

如果存在账户登录问题，请确认您是否已启用浏览器 cookie。在登录期间，本网站使用 cookie 在您的计算机上（以加密格式）存储您的登录 ID 和密码。

要启用 cookie，请根据您的浏览器版本选择对应的指示说明，并进行操作。

Binary (C.R.) S.A.

Chrome 51+

  • 前往 "Chrome 菜单(自定义和控制Google Chrome)" 的下拉选项并选择设置。
  • 点击显示高级设置。
  • 点击位于隐私菜单下方的内容设置按钮。
  • 选择允许设置局部数据（推荐）。
  • 点击完成按钮保存设置。
Binary (IOM) Ltd

Mozilla Firefox 47+

  • 打开菜单并选择首选项。
  • 点击隐私选项卡。
  • 将"Firefox will" 设置为 "记住历史"。
  • 点击OK按钮保存设置。
Binary (Europe) Ltd

Safari 9.x+

  • 前往Safari菜单并选择首选项。
  • 点击隐私选项卡。
  • 将"Block cookies:" 设置为永不。
  • 点击右上方的X按钮保存并关闭。
Binary (C.R.) S.A.

Opera 38+

  • 前往工具菜单并选择首选项。
  • 点击高级选项卡。
  • 从左列选择Cookies。
  • 选择接受cookies。
  • 点击OK按钮保存并关闭。
Binary Ltd

Microsoft Internet Explorer 11+

  • 前往工具菜单并选择Internet 选项。
  • 点击隐私选项卡。
  • 点击高级按钮。
  • 选择重写自动cookie处理。
  • 选择接受第一方Cookies与第三方Cookies。
  • 点击OK按钮保存并关闭。
Binary Ltd

Microsoft Edge 12+

  • 前往进一步操作菜单并选择设置。
  • 前往高级设置，点击查看高级设置 button.按钮。
  • 在Cookies分区选择不要阻止cookies选项。
  • 在菜单外任何地方点击保存并关闭。
