Login Problem

Sollten beim Einloggen in Ihr Konto Schwierigkeiten auftreten, überprüfen Sie bitte, ob in Ihrem Browser Cookies aktiviert sind. Diese Website verwendet Cookies, um Ihre Log-in ID und Passwort (im verschlüsselten Format) während der Log-in Session auf Ihrem PC zu speichern.

Um Cookies zuzulassen, befolgen Sie die nachfolgenden Anweisungen für die von Ihnen verwendete Browserversion.

Binary (C.R.) S.A.

Chrome 51+

  • Gehen Sie zum "Chrome-Menü (Google Chrome anpassen und kontrollieren)" Dropdown-Menü und wählen Sie Einstellungen aus.
  • Klicken Sie auf Erweiterte Einstellungen anzeigen.
  • Klicken Sie auf die Inhaltseinstellungen Schaltfläche unter dem Datenschutz Menü.
  • Wählen Sie Lokale Daten festlegen erlauben (empfohlen) aus.
  • Klicken Sie auf die Erledigt Schaltfläche, um Ihre Einstellungen zu speichern.
Binary (IOM) Ltd

Mozilla Firefox 47+

  • Menü öffnen und Einstellungen auswählen.
  • Klicken Sie auf den Datenschutz Tab.
  • Stellen Sie "Firefox wird" auf "Verlaufsgeschichte erinnern" ein.
  • Klicken Sie auf die OK Schaltfläche, um Ihre Einstellungen zu speichern.
Binary (Europe) Ltd

Safari 9.x+

  • Rufen Sie das Menü Safari auf und wählen Sie Einstellungen aus.
  • Klicken Sie auf den Datenschutz Tab.
  • Stellen Sie Cookies blockieren: auf Nie ein.
  • Klicken Sie auf die X Schaltfläche rechts oben, um zu speichern und zu schließen.
Binary (C.R.) S.A.

Opera 38+

  • Rufen Sie das Menü Tools auf und wählen Sie Einstellungen aus.
  • Klicken Sie auf den Erweitert Tab.
  • Wählen Sie Cookies aus der linken Spalte aus.
  • Wählen Sie Cookies akzeptieren aus.
  • Klicken Sie auf die OK Schaltfläche, um zu speichern und zu schließen.
Binary Ltd

Microsoft Internet Explorer 11+

  • Rufen Sie das Menü Tools auf und wählen Sie Internetoptionen aus.
  • Klicken Sie auf den Datenschutz Tab.
  • Klicken Sie auf die Erweitert Schaltfläche.
  • Wählen Sie Automatische Cookie-Behandlung überschreiben aus.
  • Wählen Sie Annehmen für beide First-Party-Cookies und Third-Party-Cookies aus.
  • Klicken Sie auf die OK Schaltfläche, um zu speichern und zu schließen.
Binary Ltd

Microsoft Edge 12+

  • Rufen Sie das Menü Weitere Aktionen auf und wählen Sie Einstellungen aus.
  • Klicken Sie unter Erweiterte Einstellungen auf die Schaltfläche "Erweiterte Einstellungen ansehen".
  • Wählen Sie unter dem Abschnitt "Cookies" die Option "Keine Cookies blockieren" aus.
  • Klicken Sie irgendwo außerhalb des Menüs, um zu speichern und zu schließen.
