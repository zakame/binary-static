Masalah Login

Jika Anda mengalami kesulitan log in ke account Anda, mohon pastikan bahwa cookie telah diaktifkan pada browser Anda. Website ini menggunakan cookies untuk menyimpan ID login dan password Anda (dalam format sandi) pada PC Anda selama sesi login.

Untuk mengaktifkan cookie, ikuti petunjuk di bawah ini untuk versi browser yang anda gunakan.

Binary (C.R.) S.A.

Chrome 51+

  • Kunjungi pilihan menu "Chrome Menu (Customize and control Google Chrome)" kemudian pilih Settings.
  • Klik pada Tampilkan susunan lanjutan.
  • Klik pada tombol Pengaturan konten dibawah menu Privasi.
  • Pilih Ijinkan pemasangan data lokal (direkomendasikan).
  • Klik pada tombol Done untuk menyimpan pengaturan Anda.
Binary (IOM) Ltd

Mozilla Firefox 47+

  • Buka menu dan pilih Preferensi.
  • Klik pada tab Pribadi.
  • Pilih "Firefox will" ke "Remember history".
  • Klik pada tombol OK untuk menyimpan pengaturan Anda.
Binary (Europe) Ltd

Safari 9.x+

  • Kunjungi menu Safari dan pilih Preferences.
  • Klik pada tab Pribadi.
  • Pilih "Block cookies:" ke Never.
  • Klik pada tombol X pada kanan atas untuk menyimpan dan menutup.
Binary (C.R.) S.A.

Opera 38+

  • Kunjungi menu Tools dan pilih Preferences.
  • Klik pada tab Advanced.
  • PilihCookies dari kolom kiri.
  • Pilih Accept cookies.
  • Klik pada tombol OK untuk menyimpan dan menutup.
Binary Ltd

Microsoft Internet Explorer 11+

  • Kunjungi menu Tools dan pilih Internet Options.
  • Klik pada tab Pribadi.
  • Klik pada tombol Lanjutan.
  • Pilih Override automatic cookie handling.
  • Pilih Accept untuk kedua First-party Cookies dan Third-party Cookies.
  • Klik pada tombol OK untuk menyimpan dan menutup.
Binary Ltd

Microsoft Edge 12+

  • Kunjungi menu More actions dan pilih Settings.
  • Dibawah Pengaturan Lanjut, klik pada tombol Lihat pengaturan lanjut.
  • Dibawah bagian Cookies pilih opsi "Don't block cookies".
  • Klik dimana saja diluar menu untuk menyimpan dan menutup.
