Remporter le contrat

Le règlement des options asiatiques se base sur une comparaison entre le dernier tick et le prix spot moyen sur la période d'activité de l'option.

Si vous sélectionnez « Asiatique/hausse », vous gagnez si le dernier tick est supérieur à la moyenne des ticks.

Si vous sélectionnez « Asiatique/baisse », vous gagnez si le dernier tick est inférieur à la moyenne des ticks.

If the last tick is equal to the average of the ticks, you don't win the payout.

Remporter le contrat

Si vous sélectionnez « Égal », vous gagnez si le dernier chiffre du dernier tick est égal à votre prédiction.

Si vous sélectionnez « Différent », vous gagnez si le dernier chiffre du dernier tick n'est pas égal à votre prédiction.

Remporter le contrat

Si vous sélectionnez « Termine dans la zone », vous gagnez si le point de sortie est strictement supérieur à la barrière inférieure ET strictement inférieur à la barrière supérieure.

Si vous sélectionnez « Termine hors de la zone », vous gagnez si le point de sortie est SOIT strictement supérieur à la barrière supérieure, OU strictement inférieur à la barrière inférieure.

Si la sortie spot est égal soit à la barrière basse ou à la barrière haute, vous ne gagnez pas le paiement.

Remporter le contrat

Si vous sélectionnez « Pair », vous gagnez si le dernier chiffre du dernier tick est un nombre pair (par ex. 2, 4, 6, 8, ou 0).

Si vous sélectionnez « Impair », vous gagnez si le dernier chiffre du dernier tick est un nombre impair (par ex. 1, 3, 5, 7, ou 9).

Remporter le contrat

Si vous sélectionnez « Supérieur », vous gagnez si le point de sortie est strictement supérieur à la barrière.

If you select "Higher", you win the payout if the exit spot is higher than the barrier.

Si vous sélectionnez « Inférieur », vous gagnez si le point de sortie est strictement inférieur à la barrière.

If the exit spot is equal to the barrier, you only win the payout for "Higher" contracts

Si la sortie spot est égale à la barrière, vous ne gagnez pas le paiement.

Remporter le contrat

Si vous sélectionnez « Au-dessus », vous gagnez si le dernier chiffre du dernier tick est supérieur à votre prédiction.

Si vous sélectionnez « En dessous », vous gagnez si le dernier chiffre du dernier tick est inférieur à votre prédiction.

Remporter le contrat

Si vous sélectionnez « Hausse », vous gagnez si le point de sortie est strictement supérieur au point d'entrée.

Si vous sélectionnez « Baisse », vous gagnez si le point de sortie est strictement inférieur au point d'entrée.

If the exit spot is equal to the entry spot, you don't win the payout.

Remporter le contrat

Vous remportez le prix si le prix du marché se termine par le chiffre que vous avez sélectionné.

Remporter le contrat

Vous remportez le prix si le prix du marché ne se termine pas par le chiffre que vous avez sélectionné.

Remporter le contrat

Si vous sélectionnez « Hausse », vous gagnez si le prix du marché est supérieur au point d'entrée.

Si vous sélectionnez « Baisse », vous gagnez si le prix du marché est inférieur au point d'entrée.

Remporter le contrat

Si vous sélectionnez « Reste dans la zone », vous gagnez si le marché ne touche jamais ni la barrière supérieure ni la barrière inférieure, sur toute la durée de la période de contrat.

Si vous sélectionnez « Sort de la zone », vous gagnez si le marché touche soit la barrière supérieure, soit la barrière inférieure, à tout moment de la période de contrat.

Remporter le contrat

Si vous sélectionnez « Hausse », vous gagnez si le point de sortie est strictement supérieur au point d'entrée.

Si vous sélectionnez « Baisse », vous gagnez si le point de sortie est strictement inférieur au point d'entrée.

Remporter le contrat

Si vous sélectionnez « Touche », vous gagnez si le marché touche la barrière à tout moment de la période de contrat.

Si vous sélectionnez « Ne touche pas », vous gagnez si le marché ne touche jamais la barrière sur toute la durée de la période de contrat.

Remporter le contrat

Si vous sélectionnez « Hausse », vous gagnez si le prix du marché est supérieur au point d'entrée.

Si vous sélectionnez « Baisse », vous gagnez si le prix du marché est inférieur au point d'entrée.

Versement

En achetant le contrat Clotûre-Bas, vous gagnerez plusieurs fois la différence entre la %clotûre et le plus bas pendant toute la durée du contrat.

Versement

En achetant le contrat «Haut-Clotûre», vous gagnerez plusieurs fois la différence entre le plus haut et la clotûre pendant toute la durée du contrat.

Versement

En achetant le contrat «Haut-Bas», vous gagnerez plusieurs fois la différence entre le plus haut et le plus bas pendant toute la durée du contrat.
