ログインにお困りの場合

ご口座へのログインが上手くいかない場合は、ブラウザの設定からCookieを有効に設定して下さい。当社サイトでは、ログイン時にお客さまのログインIDと暗号化されたパスワードをお客さまのコンピュータに保存する目的でCookieを利用しています。

ご利用中のブラウザのバージョンの指示に従ってクッキーを有効化して下さい。

Binary (C.R.) S.A.

Chrome 51+

  • "Chromeメニュー（Google Chromeのカスタマイズ及び管理）"へいき、ドロップダウンにある設定を選択してください。
  • 高度な設定を表示するをクリックしてください。
  • プライバシーメニューにあるコンテンツ設定をクリックしてください。
  • ローカルへのデータ設定を許可する（推奨）を選択してください。
  • ”完了”ボタンをクリックし、現在の設定を保存して下さい。
Binary (IOM) Ltd

Mozilla Firefox 47+

  • メニューを開いて設定を選択してください。
  • プライバシータブをクリックしてください。
  • "Firefox"を"履歴を記憶する"に設定してください。
  • OKボタンを押してセーブし閉じて下さい。
Binary (Europe) Ltd

Safari 9.x+

  • Safariメニューへいき、設定を選択してください。
  • プライバシータブをクリックしてください。
  • ”Cookieをブロック”をNeverに設定してください。
  • 保存をして閉じるには、右上にあるXボタンをクリックしてください。
Binary (C.R.) S.A.

Opera 38+

  • ツールメニューへいき、設定を選択してください。
  • 高度な設定タブをクリックしてください。
  • 左の欄にあるCookieを選択してください。
  • Cookieを有効にするを選択してください。
  • OKボタンを押してセーブし閉じて下さい。
Binary Ltd

Microsoft Internet Explorer 11+

  • ツールメニューへいき、インターネットオプションを選択してください。
  • プライバシータブをクリックしてください。
  • 高度な設定ボタンをクリックしてください。
  • 自動Cookie処理を上書きするを選択してください。
  • ファーストパーティCookieとサードパーティCookieを許可するを選択してください。
  • OKボタンを押してセーブし閉じて下さい。
Binary Ltd

Microsoft Edge 12+

  • その他の設定メニューの設定を選択してください。
  • 設定にある高度な設定を表示ボタンをクリックしてください。
  • Cookieセクションにある"Cookieをブロックしない"オプションを選択してください。
  • 保存をして閉じるには、メニュー外をクリックしてください。
