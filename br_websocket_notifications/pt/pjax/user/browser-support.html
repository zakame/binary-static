Problema com Login

Se estiver tendo dificuldades para conectar-se à sua conta, verifique se os cookies estão ativados no seu browser. Este website utiliza cookies para guardar o seu login e senha (em formato encriptado) no seu PC durante a sua sessão de login.

Para ativar os cookies, siga as instruções abaixo para a versão de browser que está a utilizar.

Binary (C.R.) S.A.

Chrome 51+

  • Vá para o "Menu suspenso Chrome (Personalize e controle Google Chrome)" selecione Configurações.
  • Clique em Mostrar configurações avançadas.
  • Clique no botão Configurações de contúdo sob o menu Privacidade.
  • Selecione Permitir a configuração de dados locais (recomendado).
  • Clique no botão Concluído para salvar as suas configurações.
Binary (IOM) Ltd

Mozilla Firefox 47+

  • Abra o menu e selecione Preferências.
  • Clique na guia Privacidade.
  • Defina "Firefox irá" para "Memorizar histórico".
  • Clique no botão OK para salvar as suas configurações.
Binary (Europe) Ltd

Safari 9.x+

  • Vá para o menu Safari e selecione Preferências.
  • Clique na guia Privacidade.
  • Defina "Bloquear cookies:" para Nunca.
  • Clique no botão X na parte superior direita para salvar e fechar.
Binary (C.R.) S.A.

Opera 38+

  • Vá para o menu Ferramentas e selecione Preferências.
  • Clique na guia Avançado.
  • Selecione Cookies da coluna esquerda.
  • Selecione Aceitar cookies.
  • Clique no botão OK para salvar e fechar.
Binary Ltd

Microsoft Internet Explorer 11+

  • Vá para o menu Ferramentas e selecione Opções de Internet.
  • Clique na guia Privacidade.
  • Clique no botão Avançado.
  • Selecione Ignorar manipulação automática de cookies.
  • Selecione Aceitar tanto para Cookies de primeiros e Cookies de terceiros.
  • Clique no botão OK para salvar e fechar.
Binary Ltd

Microsoft Edge 12+

  • Vá para o menu Mais ações e selecione Configurações.
  • Sob Configurações avançadas, clique no botão Ver configurações avançadas.
  • Na seçãoCookies, selecione a opção "Não bloquear cookies".
  • Clique em qualquer sítio fora do menu para salvar e fechar.
