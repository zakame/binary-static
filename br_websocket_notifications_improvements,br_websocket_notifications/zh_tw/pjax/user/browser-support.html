登入故障

如果存在帳戶登入問題，請確認您的瀏覽器cookie 是否啟用。在登入時域期間，本網站使用cookie 在您的電腦上（以加密格式）儲存您的登入ID 和密碼。

要啟用cookie，請按下面符合您瀏覽器版本的指示說明進行操作。

Binary (C.R.) S.A.

Chrome 51+

  • 前往"Chrome 功能表 (自訂與控制Google Chrome)"下拉選項並選擇設定。
  • 點擊顯示進階設定。
  • 點擊位於隱私選項卡下方的內容設定按鈕。
  • 選擇允許設定本機資料(推荐)。
  • 點擊完成按鈕儲存設定。
Binary (IOM) Ltd

Mozilla Firefox 47+

  • 打開選項卡並選擇喜好設定。
  • 點擊隱私標籤。
  • 將"Firefox will"設定為 "記住歷史"。
  • 點擊OK按鈕儲存設定。
Binary (Europe) Ltd

Safari 9.x+

  • 前往Safari選項卡並選擇喜好設定。
  • 點擊隱私標籤。
  • 將"Block cookies:"設定為永不。
  • 點擊右上方的X按鈕儲存並關閉。
Binary (C.R.) S.A.

Opera 38+

  • 前往工具選項卡並選擇喜好設定。
  • 點擊進階標籤。
  • 從左欄選擇Cookies設定。
  • 選擇接受cookies。
  • 點擊OK按鈕儲存並關閉。
Binary Ltd

Microsoft Internet Explorer 11+

  • 前往工具選項卡並選擇Internet選項。
  • 點擊隱私標籤。
  • 點擊進階按鈕。
  • 選擇覆寫自動cookie處理。
  • 選擇接受第一方Cookies與第三方Cookies。
  • 點擊OK按鈕儲存並關閉。
Binary Ltd

Microsoft Edge 12+

  • 前往進一步操作選項卡並選擇設定。
  • 前往進階設定，點擊檢視進階設定按鈕。
  • 在Cookies分區選擇不要阻止cookies選項。
  • 在功能表外任何地方點擊儲存並關閉。
